# Rust Playground

> 原文：[https://freddiehaddad.github.io/fast-track-to-rust/rust_playground.html](https://freddiehaddad.github.io/fast-track-to-rust/rust_playground.html)

In addition to running and editing code in the browser, you might find the [Rust Playground](https://play.rust-lang.org/) helpful. It's a web interface for running Rust code.

```rs
fn main() {
    println!("Welcome to Fast Track to Rust!");
}
```

# [Exercise](#exercise)

*   Hover the mouse over the code block and click the arrow button to run the code.
*   Modify the `"Welcome to Fast Track to Rust!"` string and re-run the code.
*   Revert your changes by hovering your mouse over the code block and clicking the *Undo changes* button.
*   Run the hello world program in the [playground](https://play.rust-lang.org/?version=stable&mode=debug&edition=2021&code=fn+main%28%29+%7B%0A++++println%21%28%22Welcome+to+Fast+Track+to+Rust%21%22%29%3B%0A%7D).
