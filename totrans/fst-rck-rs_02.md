# Shortcuts

> 原文：[https://freddiehaddad.github.io/fast-track-to-rust/shortcuts.html](https://freddiehaddad.github.io/fast-track-to-rust/shortcuts.html)

This course, developed with [mdbook](https://github.com/rust-lang/mdBook), features efficient keyboard shortcuts for streamlined navigation.

## [Keyboard Shortcuts](#keyboard-shortcuts)

*   `Arrow-Left`: Navigate to the previous page.
*   `Arrow-Right`: Navigate to the next page.
*   `s`: Activate the search bar.
*   `Ctrl+Enter`: Execute the code in the focused code block.
