# Using Cargo

> 原文：[https://freddiehaddad.github.io/fast-track-to-rust/using_cargo.html](https://freddiehaddad.github.io/fast-track-to-rust/using_cargo.html)

In this chapter we'll cover using [cargo](https://doc.rust-lang.org/book/ch01-03-hello-cargo.html) to create a package, compile the boilerplate code, and run the program. The package we create will be used for the remainder of the book to build our rustle program.
