## Rustle 功能

Rustle 是一个功能相当丰富的程序。在本课程中，我们不会实现所有功能。相反，我们将专注于一组功能（在行为上略有变化），这将帮助你学习 Rust。我们将实现的功能包括：

| 命令行参数 | 描述 |
| --- | --- |
| `-n`, `--line-number` | 在输出文件的每一行前加上基于 1 的行号。 |
| `-A num`, `--after-context=num` | 打印匹配行之后 *num* 行的上下文。 |
| `-B num`, `--before-context=num` | 在匹配行之前打印 *num* 行的上下文。 |

## 打印行号

我们将要添加到 rustle 程序中的第一个功能是打印匹配我们模式的任何行的行号。如果你来自像 C/C++ 这样的语言，你可能倾向于使用一个变量来处理这个问题，并在每次循环迭代中递增它。虽然这可以工作，但它引入了像 C 和 C++ 这样的语言中存在的问题（例如，越界索引、整数溢出等）。我们将演示这一点，并随后介绍一种更符合习惯的方法。让我们开始吧！
