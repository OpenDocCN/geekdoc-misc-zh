# 快捷键

> 原文：[`freddiehaddad.github.io/fast-track-to-rust/shortcuts.html`](https://freddiehaddad.github.io/fast-track-to-rust/shortcuts.html)

本课程由 [mdbook](https://github.com/rust-lang/mdBook) 开发，提供了高效的键盘快捷键以实现流畅的导航。

## 键盘快捷键

+   `箭头左`: 导航到上一页。

+   `箭头右`: 导航到下一页。

+   `s`: 激活搜索栏。

+   `Ctrl+Enter`: 执行焦点代码块中的代码。
