# 依赖项

> 原文：[`freddiehaddad.github.io/fast-track-to-rust/dependencies.html`](https://freddiehaddad.github.io/fast-track-to-rust/dependencies.html)

[crates.io](https://crates.io/) 是 Rust 社区的中央[包注册中心](https://doc.rust-lang.org/cargo/appendix/glossary.html#registry)，作为发现和下载包的中心，`cargo`支持为您添加和管理[依赖项](https://doc.rust-lang.org/cargo/guide/dependencies.html)，并且默认配置为使用[包注册中心](https://doc.rust-lang.org/cargo/appendix/glossary.html#registry)。

在本课程中，我们将探索几个 crate：

+   **[regex](https://crates.io/crates/regex)**: Rust 的正则表达式实现

+   **[itertools](https://crates.io/crates/itertools)**: 额外的迭代器适配器、迭代器方法、自由函数和宏

+   **[clap](https://crates.io/crates/clap)**: 一个简单易用、高效且功能齐全的命令行参数解析器
