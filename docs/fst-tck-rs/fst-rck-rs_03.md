# 安装

> 原文：[`freddiehaddad.github.io/fast-track-to-rust/installation.html`](https://freddiehaddad.github.io/fast-track-to-rust/installation.html)

本书中的大多数练习都可以在您的网络浏览器中完成，但有少数例外。然而，建议在您的系统上安装 Rust 开发工具链。一旦安装完成，您将拥有构建和本地运行 Rust 代码所需的一切。

安装是通过使用 `rustup`，Rust 工具链安装程序来完成的。一旦完成，您将拥有 `rustup`、`cargo` 和 `rustc`。

按照位于 [`rustup.rs/`](https://rustup.rs/) 的说明完成安装。

## 配置您的 IDE

建议但不是必需的，将您的编辑器配置为与 Rust 一起使用。安装 Cargo 后，大多数编辑器，如 VS Code，将自动安装 [rust-analyzer](https://rust-analyzer.github.io/)，这是 Rust 编程语言的 LSP 实现。
