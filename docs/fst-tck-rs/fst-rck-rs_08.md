# 语言基础

> 原文：[`freddiehaddad.github.io/fast-track-to-rust/language_basics.html`](https://freddiehaddad.github.io/fast-track-to-rust/language_basics.html)

Rust 包含了您已经熟悉的原始类型、^(1) 语句和^(2) 表达式。然而，它还融合了一些来自函数式语言的有力概念，这些概念我们将在本课程中探讨。为了开始我们的 rustle 程序，我们将保持简单，并逐步重构代码以介绍 Rust 提供的更多功能。让我们从变量开始。

* * *

1.  Rust 内置类型的相关详细信息可以在官方 [Rust 文档](https://doc.rust-lang.org/reference/types.html) 中找到。↩

1.  语言语句和表达式的详细信息可以在官方 [Rust 文档](https://doc.rust-lang.org/reference/statements-and-expressions.html) 中找到。↩
