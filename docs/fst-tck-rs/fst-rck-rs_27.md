# 包

> 原文：[`freddiehaddad.github.io/fast-track-to-rust/packages.html`](https://freddiehaddad.github.io/fast-track-to-rust/packages.html)

包是由一个或多个提供一组功能的单元组成的集合。它包含一个`Cargo.toml`文件，概述了如何构建这些单元。一个包可以包含多个二进制单元，但最多只能有一个库单元。然而，一个包必须至少包含一个单元，无论是库单元还是二进制单元。

请参考 Rust 文档中的[包和单元](https://doc.rust-lang.org/book/ch07-01-packages-and-crates.html#packages-and-crates)部分。
