# Rust Playground

> 原文：[`freddiehaddad.github.io/fast-track-to-rust/rust_playground.html`](https://freddiehaddad.github.io/fast-track-to-rust/rust_playground.html)

除了在浏览器中运行和编辑代码外，您可能还会发现 [Rust Playground](https://play.rust-lang.org/) 有用。这是一个运行 Rust 代码的网页界面。

```rs
fn main() {
    println!("Welcome to Fast Track to Rust!");
}
```

# 练习

+   将鼠标悬停在代码块上并点击箭头按钮以运行代码。

+   修改字符串 `"Welcome to Fast Track to Rust!"` 并重新运行代码。

+   通过将鼠标悬停在代码块上并点击 *撤销更改* 按钮来撤销您的更改。

+   在 [playground](https://play.rust-lang.org/?version=stable&mode=debug&edition=2021&code=fn+main%28%29+%7B%0A++++println%21%28%22Welcome+to+Fast+Track+to+Rust%21%22%29%3B%0A%7D) 中运行 hello world 程序。
