# 前言

## 为什么要学习编程语言？

在开始一门课程学习之前询问为什么我们应该参与其中总是有益的。因此，让我们列出一些理由来证明这一点：

+   编程语言是人类发明的一些最美丽、最强大的对象。

+   它们让我们——实际上，迫使我们——精确地捕捉我们的思想，并以此命令强大的机器为我们执行任务。

+   对于程序员来说，它们是终极的人机界面。

+   它们在工业上非常重要：重要到主要公司会创建新的语言或采用现有的语言来建立或扩大他们的市场份额。

+   它们体现了一种新的知识形式：[《计算机程序的构造和解释》](https://www.google.com/url?q=https://mitp-content-server.mit.edu/books/content/sectbyfn/books_pres_0/6515/sicp.zip/index.html&sa=D&source=editors&ust=1695232021060331&usg=AOvVaw0FxIsmtPtHl1wGbU99LU18) 所称的“程序性认识论”。

+   它们使我们能够证明关于我们表达手段和限制的强大论断。还有多少其他语言让我们做到这一点？
