# 教你自己 SMoL

这项工作围绕你对 SMoL 的理解。然而，为了避免被动的阅读体验，[黄陈晨](https://www.google.com/url?q=https://lukc1024.github.io/&sa=D&source=editors&ust=1695232021072512&usg=AOvVaw3O6JByCXbnLV7Fth_iekCw) 实施了一系列自我定速的教程来教授 SMoL。这些教程既提供了概念知识，又教授了重要的术语。重要的是，它们还包含简短、快速的问题测试，以确保你走在正确的道路上。这些问题有两个目的。首先，它们迫使你注意教程：你不能只是被动地滚动浏览。其次，它们基于对这一材料的已知误解。其他学习者犯了这些错误，你也可能犯。请注意！

这些教程仍然是初步的，但它们已经在布朗大学成功使用。

由于当前教程不支持停止、保存工作并继续，我们将材料分解成一系列小(!)教程，每个教程涵盖一个概念（有时甚至更少）。这样，你可以完成一个或三个教程，休息一下，然后继续。
