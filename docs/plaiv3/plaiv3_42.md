## 语言中的函数

有许多方法可以思考向语言中添加函数。例如，许多语言都有顶级函数；例如：

```
fun f(x):
  x + x
```

事实上，一些语言（如 C 语言）只有顶级函数。然而，大多数现代语言都有在顶级之外编写函数的能力：例如，

```
fun f(x):
  fun sq(y):
    y * y
  sq(x) + sq(x)
```

甚至可以返回这些函数，甚至允许它们以匿名方式编写。由于几乎每种现代语言都支持它，我们将将其视为 SMoL 的一个组成部分。实际上，有了这样的功能，我们实际上并不真的需要一个命名的函数构造：我们反而可以写成

```
fun f(x):
  sq = lam(y): y * y
  sq(x) + sq(x)
```

并且反过来，我们也可以用名称绑定和`lam`来替换`f`。
