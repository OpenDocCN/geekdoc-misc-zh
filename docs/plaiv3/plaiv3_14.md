## 替换

顺便说一下，注意你不需要了解任何计算机编程知识就能回答这些问题。你在中学和高中代数课上已经做过类似的事情了。你可能学过“替换”这个短语。这就是我们在这里遵循的过程。实际上，我们可以把编程看作是代数的一个自然延伸，只不过数据类型更加有趣：不仅包括数字，还有字符串、图像、列表、表格、矢量场、视频等等。

好的，这样我们就有了实现一个评估器的方法：

+   找一种方法来表示程序源代码（例如，一个字符串或一棵树）。

+   寻找下一个要评估的表达式。

+   进行替换（文本上）以获得一个新的程序。

+   继续评估，直到只剩下值为止。

然而，正如你可能猜到的，大多数编程语言实际上并不是这样工作的：通常这会非常慢。所以，我们需要找到一种更好的方法！
