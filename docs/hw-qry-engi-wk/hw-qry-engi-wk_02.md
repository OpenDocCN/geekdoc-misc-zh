# 前言

> [原文：https://howqueryengineswork.com/00-preface.html](https://howqueryengineswork.com/00-preface.html)

本书的第一版是在工作之间的一个星期假期中撰写的，并在 2020 年的 COVID 封锁期间得到改进。我老实说，我不确定有多少人想读一本关于从头开始构建查询引擎的书。结果证明，相当多的人感兴趣。

第二版是一个重大修订，具有更清晰的解释、更好的代码组织和对连接、子查询和分布式查询执行等主题的扩展覆盖。

本书不是学术性质的。它旨在成为一本实用、直接的指南。读者的反馈令人鼓舞，所以感谢所有提供建议和纠正的人。我希望这一版能成为理解查询引擎的更好基础。
