# 基本计算器（A Basic Calculator）

> 原文：[`rust-exercises.com/100-exercises/02_basic_calculator/00_intro.html`](https://rust-exercises.com/100-exercises/02_basic_calculator/00_intro.html)

在本章中，我们将学习如何将 Rust 用作**计算器**。

这可能听起来不多，但它将给我们一个机会来覆盖 Rust 的基础知识，例如：

+   如何定义和调用函数

+   如何声明和使用变量

+   基本类型（整数和布尔值）

+   算术运算符（包括溢出和下溢行为）

+   比较运算符

+   控制流

+   潜在错误（Panics）

通过几个练习来掌握基础知识，可以让语言在你的指尖流淌。当我们转向更复杂的话题，例如特性和所有权时，你将能够专注于新概念，而不会因为语法或其他琐碎细节而陷入困境。
