# 后记

> 原文：[`rust-exercises.com/100-exercises/going_further.html`](https://rust-exercises.com/100-exercises/going_further.html)

我们的 Rust 之旅到此结束。

这已经相当广泛了，但绝不是详尽的：Rust 是一个具有广阔表面区域和更大生态系统的语言！

尽管如此，不要害怕：**没有必要学习一切**。你将在项目开发过程中学习到在特定领域（后端、嵌入式、CLIs、GUIs 等）中有效所需的任何内容。

最后，没有捷径：如果你想精通某件事，你需要反复去做。在整个课程中，你已经写了不少 Rust 代码，足以让语言及其语法在你的指尖流畅运行。但要真正感到这是“你的”，还需要更多的代码行数，但只要你继续练习，这一刻一定会到来。

## 进一步学习

让我们以一些指向你可能在你继续 Rust 之旅时发现有用的额外资源的指针来结束。

### 练习

你可以在 `rustlings` 项目和 [exercism.io](https://exercism.io)'s Rust 轨道上找到更多练习 Rust 的练习。

### 入门材料

如果你想要从不同的角度了解我们在这门课程中涵盖的相同概念，请查看 [Rust 书籍](https://doc.rust-lang.org/book/title-page.html) 和 ["Programming Rust 第二版"](https://www.oreilly.com/library/view/programming-rust-2nd/9781492052586/)。你肯定会学到一些新东西，因为它们并没有涵盖完全相同的话题；Rust 有很多表面区域！

### 高级材料

如果你想更深入地了解这门语言，请参考 [Rustonomicon](https://doc.rust-lang.org/nomicon/) 和 ["Rust for Rustaceans"](https://nostarch.com/rust-rustaceans)。

["去锈"系列](https://www.youtube.com/playlist?list=PLqbS7AVVErFirH9armw8yXlE6dacF-A6z) 是学习许多最受欢迎的 Rust 库内部结构的另一个优秀资源。

### 特定领域材料

如果你想要使用 Rust 进行后端开发，请查看 ["Rust 从零到生产"](https://zero2prod.com)。

如果你想要使用 Rust 进行嵌入式开发，请查看 [嵌入式 Rust 书籍](https://docs.rust-embedded.org/book/)。

### 大师班

你可以找到跨领域的关键主题资源。

为了测试，请查看 ["高级测试，超越基础"](https://rust-exercises.com/advanced-testing/)。

对于遥测，请查看 ["你无法修复你看不见的东西"](https://rust-exercises.com/telemetry/)。
