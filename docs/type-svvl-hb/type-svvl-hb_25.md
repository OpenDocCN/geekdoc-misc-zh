# 阅读并学习 TypeScript

> 原文：[`typescriptbook.jp/reference`](https://typescriptbook.jp/reference)

本章では，TypeScript 的语言功能将被逐一解释。解释的范围不仅限于 TypeScript，还包括了 JavaScript 中必要的内容。这是因为许多学习 TypeScript 的人也同时学习 JavaScript。

我们特意考虑到了哪些功能属于 TypeScript，哪些属于 JavaScript。对于 JavaScript 的功能，我们会使用"JavaScript 的~"，而对于 TypeScript 的功能，我们会使用"TypeScript 的~"等语言名称来在文中标明。虽然语言名称会频繁出现，可能会让人觉得烦人，但清楚地理解哪些功能属于哪种语言是很重要的，希望您能耐心一点。

本章的内容从代码中的小部分开始，逐渐深入到更大的部分。从值和变量开始，然后是语句、函数，以及面向对象编程元素如类等，进一步讨论模块等更大的粒度的元素。
